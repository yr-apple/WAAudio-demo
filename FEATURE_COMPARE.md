## ❌ 未实现功能 (优先级排序)

### P0 - 核心功能
1. **UI 集成** - 波形编辑 UI
2. **标记点 UI** - 标记点显示和交互

### P1 - 重要功能
1. **门限 (Gate)** - 降噪门
2. **限制器 (Limiter)** - 峰值限制
3. **多轨录音** - 同时录音多个轨道
4. **发送/返回效果** - 效果路由

### P2 - 高级功能
1. **变速不变调** - 时间拉伸
2. **变调** - 音高调整
3. **消除人声** - 相位抵消
4. **降噪** - 噪声门 + 降噪算法
5. **自动化** - 参数自动化包络

### P3 - 专业功能
1. **相位分析** - 立体声相位检查
2. **响度分析** - LUFS/LKFS
3. **批次处理** - 批量音频处理
4. **CD 刻录** - 音频 CD 制作
5. **视频同步** - 音视频对齐
